# Project DAAN - FERN Stack
Digital Analytics for Asset-based Navigation of Roads

## Tech Stack
- **F**astAPI (Backend)
- **E**xpress patterns (API structure)
- **R**eact + Vite (Frontend)
- **N**ode.js (Runtime)

Plus:
- Supabase (PostgreSQL)
- Cloudflare R2 (File Storage)

## Quick Start

### Backend
\\ash
cd server
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
\
### Frontend
\\ash
cd client
npm install
npm run dev
\
## Deployment
- Frontend: Vercel
- Backend: Render
